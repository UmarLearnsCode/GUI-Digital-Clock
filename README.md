# GUI-Digital-Clock
Responsive Digital Clock | GUI Project. Live time display.

**About:**
A desktop application built using Python and Tkinter library. This GUI-based project provides a clean and user-friendly interface with real-time functionality. No external libraries or frameworks were used - only core Python and Tkinter for all UI elements and logic. Lightweight, fast, and runs on Windows, Mac & Linux without any dependencies.

**Key Features:** 
1. Fully GUI-based using Tkinter - no command line
2. Real-time updates and user interactions 
3. Clean interface with buttons, labels, and input fields
4. Cross-platform compatibility

**Tech Stack:** 
Python 3.x, Tkinter
